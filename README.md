# Lab 13 — Коллекции и структуры данных (JavaScript)

## Тема
Работа со встроенными коллекциями JavaScript (Array и Map).

## Цель работы
Изучить динамические структуры данных и применить их в веб-приложении.

## Функционал
- добавление элементов в коллекцию;
- удаление элементов;
- поиск по имени;
- отображение всех данных;
- сортировка по имени.

## Используемые коллекции
- `Array` — хранение списка студентов
- `Map` — хранение дополнительной информации (дата добавления)
